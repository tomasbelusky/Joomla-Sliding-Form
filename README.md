Joomla Sliding Form
===================

Easy customizable sliding form for Joomla! 2.5 and 3.x

Base parameters
---------------
  - recipient's email
  - (don't) show or require name field
  - (don't) show or require subject field
  - default subject

Advanced parameters
-------------------
  - vertical alignment
  - horizontal alignment
  - width
  - border radius
  - font color
  - background color
  - button text
